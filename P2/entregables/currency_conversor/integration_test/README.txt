HERRAMIENTAS

tools.dart tiene una función para realizar llamadas a la API y poder comprobar
que el resultado que aparece en la aplicación es el correspondiente a la tasa
de conversión en el momento actual. Decidimos esta solución frente al uso del 
stub para poder ejecutar los test de comprobación de errores. 


TEST PARA MOVIL

Para los test con conexión ejecutar las pruebas en:
    widget_mobile_test.dart

Para los test sin conexion ejecutar las pruebas en:
    widget_mobile_error_test.dart
